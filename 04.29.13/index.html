CSS Zen Garden, July 2012 (Google Search)

CSS Zen Garden is a challenge to present the same (unmodified) HTML content in a unique way using only CSS. View other designs here.

This CSS Zen Garden design came to me when I was staring at a blank Google search box. I had planned on searching images for inspiration, but I couldn't think of a search term. Then I realized I could just make it look like Google search results. Problem solved.

These screenshots are side-by-side views of actual Google results (on the left) and my design (right).

Services: image optimization & creation, CSS

CSS Zen Garden, July 2012 (Matrix)

CSS Zen Garden is a challenge to present the same (unmodified) HTML content in a unique way using only CSS. View other designs here.

My inspiration for this design was "The Matrix." First, I used a movie-inspired typeface on the background in an old-school green-and-black color scheme. And, since nothing in the Matrix is as it seems, I used CSS3 animations to reveal the body text and make the headings glow on mouseover.
